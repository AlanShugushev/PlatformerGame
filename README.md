## Description:
A simple platformer game written in Pygame.

The player can move *left and right* and *jump*.

## How to run?

1. Install Python
2. Install dependencies:

pip install -r requirements.txt

3. Run the game:

python project.py

## Controls:
- *Left/Right Arrows* — move
- *Space* — jump
